# 🎲 BetConsciente

## 📖 Sobre o Projeto
O **BetConsciente** é um aplicativo robusto desenvolvido para atuar no rastreio e auxílio contra o vício em jogos online. O sistema busca fornecer ferramentas práticas de monitoramento e conscientização, permitindo que os usuários tenham maior controle sobre seu tempo e suas finanças.

## ✨ Funcionalidades Principais
* **Rastreamento de Hábitos:** Monitoramento do tempo de uso e frequência em plataformas de apostas e jogos.
* **Gestão Financeira:** Registro de entradas e saídas para controle de gastos com jogos.
* **Alertas e Limites:** Sistema de notificações para alertar o usuário quando limites pré-estabelecidos de tempo ou dinheiro são atingidos.
* **Histórico e Relatórios:** Visualização de dados e estatísticas para acompanhamento do progresso.

## 💻 Tecnologias Utilizadas
Este projeto integra diferentes linguagens para garantir eficiência e robustez:
* **Java:** Estrutura principal da aplicação e regras de negócio.
* **C++:** Módulos de alta performance e processamento em nível de sistema.
* **SQL:** Armazenamento seguro de dados, histórico de atividades e gerenciamento de usuários.

## ⚙️ Pré-requisitos
Para rodar este projeto localmente (seja no Zorin OS ou em outro ambiente), você precisará ter instalado:
* **JDK (Java Development Kit)** (versão X ou superior)
* **Compilador C++** (como o GCC/G++)
* **Servidor de Banco de Dados SQL** (ex: MySQL, PostgreSQL ou SQL Server)

## 🚀 Instalação e Execução

1. **Clone o repositório:**
   ```bash
   git clone [https://github.com/seu-usuario/betconsciente.git](https://github.com/seu-usuario/betconsciente.git)
   ```

2. **Configuração do Banco de Dados:**
   * Inicie o seu servidor SQL.
   * Execute os scripts de criação do banco e das tabelas localizados na pasta `/sql`.

3. **Compilação dos Módulos (C++):**
   ```bash
   cd src/cpp
   g++ -o processador_dados main.cpp
   ```

4. **Execução da Aplicação (Java):**
   ```bash
   # Compile e rode a classe principal ou o arquivo .jar
   java -jar betconsciente.jar
   ```
